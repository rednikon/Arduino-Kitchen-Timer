# Arduino-Kitchen-Timer
Building an Arduino-powered kitchen timer at home.


## Materials
  * Arduino UNO
  * LCD display
  * 2 - push buttons
  * Toggle switch
  * 2 - LED lights
  * 2 - 10k resistors
  * 2 - 330ohm resistors
  * 9v battery
  * Battery plug
  * Buzzer
  * Jumper wires
  * Heat shrink tubing
  * Case


## Tools
  * Dremel
  * Soldering Iron
  * Hot Glue Gun
